## PLANT TRACKER 

### OVERVIEW

Template for tracking all important bits and pieces in the plants world. 

### PROBLEM

Taking care of the plants is (for most people at least) boring and repetitive work that can easily fall out of ones scope for the day. 

### GOAL

The goal with this mini project was to create a tracker and a /sort of/ a reminder to do your deeds on a daily basis. 
It serves as a guideline and a to-do list, along with the reminder that you need to do your job.

### DESIGN

##### single HTML file created with help of Claude

In the tracker there is overall knowledge base that provides information about plants that are currently located in the open space, how to take care of them, when to water them, where to position them, and (if) any additional info.

List of current functionalities:

- general plant care profile
- water and beyond-water maintenance rules
- tracking check-box for recording daily watering
- grouped daily log with drilldown ability
- calendar autopopulating with check-box data

### TO-DO AND NEXT STEPS

After some time, there will be more than enough historical data to analyse and improve the state of plants in the open space.

Next steps, in regards to written above, would be: 

- adding detailed tracking for specific plant / pot / loc - height, sun exposure time, humidity, leaf state
- adding analysis tab - data hub for all data points in one place
- AI-enhanced improvement suggestions - train small, local AI model on collected data in combination with public weather data to create a prediction model
- prediciton model for growth, number of sun hours per period etc. in order to enhance care
- as-simple-and-as-dumb-as-possible chatbot that can be asked "did i water my plants this time 2 weeks ago" for quick insights
